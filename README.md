# Overview
The game was made with SFML library as a final course work by C++.
4 levels are available.
## Install
sudo apt-get install libsfml-dev
### Controls
- Arrows, 
- Shot - Space, 
- Big shot - LCtrl, 
- Pause - P.
#####    Main goal - defend Eagle.
